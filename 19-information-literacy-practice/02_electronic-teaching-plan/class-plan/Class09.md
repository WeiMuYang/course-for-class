## 2 DOS    
### 2.1 常用DOS命令     
#### 2.1.1 其他操作      
>清屏 cls      
```shell
cls
```
>显示及修改日期date    
```shell
date 
# 当前日期: 2019/10/20 周日
# 输入新日期: (年月日) 2019/10/19
# 客户端没有所需的特权。(需要以管理员权限运行)
```
>格式化 format      
```shell
format 〈盘符：〉[/S][/4][/Q]   # 磁盘格式化命令（不予演示）
```
>ipconfig       
```shell
ipconfig /?  # 显示帮助信息
ipconfig /all
```

**注意：**以上命令都在D盘演示，因为C修改C盘的内容需要管理员权限
